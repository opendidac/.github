OpenDidac (also known as EVAL) is a comprehensive educational platform that enables educators to create engaging assessments and conduct exams across engineering and other disciplines. With a focus on ease of use and simplicity, OpenDidac provides a seamless experience for both educators and students.

Originally developed for HEIG-VD, OpenDidac is now available to the wider educational community, offering robust tools for creating and managing online evaluations with real-time insights into student engagement and performance.

OpenDidac sets itself apart with question types tailored for software engineering and beyond:

- **Code Writing**: Students write and execute code (JavaScript, Python, Java, C, C++, Golang, etc.) in a sandboxed environment using a Monaco editor (similar to VS Code)
- **Database**: PostgreSQL-focused questions to complete queries and validate results
- **Code Reading**: Understand code and predict its output
- **Web**: CSS, HTML, and JavaScript challenges
- **Multiple Choice & True/False**: Traditional assessment formats
- **Essay**: Open-ended questions for detailed responses

4. **Finished**: Students receive grades and feedback while educators access comprehensive reports
